# 프로젝트 현황

**최종 업데이트:** 2026-03-27
**현재 버전:** v0.4.0
**배포 URL:** 미정

## 최근 변경
- Docker Compose 기반 로컬 PostgreSQL 개발 환경을 추가했습니다.
- 앱 시작 시 `GET /api/v1/health`로 DB 준비 상태를 확인하고, 미연결 시 설정 안내 페이지를 표시하도록 정리했습니다.
- 로컬 개발용 기본 DB 포트를 `5433`으로 조정해 다른 프로젝트와의 기본 포트 충돌 가능성을 낮췄습니다.
- PostgreSQL 첫 생성 시 mock 사용자와 데모 레퍼런스를 1회만 주입하는 init script를 추가했습니다.
- `README`에 `.env`, Docker DB 실행, 재초기화 절차를 반영했습니다.

| 검색, 태그 필터링, 실제 페이지네이션이 UI 목업 수준에 머물러 있음 | 중간 | 진행 예정 |
| 로그인 후 실제 브라우저에서 저장/새로고침 재조회까지 검증하는 E2E 자동화가 없음 | 중간 | 진행 예정 |
| Docker DB와 앱 실행 절차가 정리됐지만, 여전히 Go 서버와 Vite 서버는 수동으로 각각 실행해야 함 | 낮음 | 유지 |
