# 프로젝트 현황

**최종 업데이트:** 2026-03-26
**현재 버전:** v0.3.0
**배포 URL:** 미정

## 최근 변경
- 대시보드가 실제 레퍼런스 목록을 조회하도록 `GET /api/v1/references` 연동을 추가했습니다.
- 레퍼런스 추가 모달이 실제 저장 API를 호출하도록 `POST /api/v1/references` 연동을 추가했습니다.
- 대시보드에 `Loading / Empty / Error / Success` 상태를 명시적으로 표시하도록 정리했습니다.
- 저장 실패 시 입력값 유지, 에러 메시지 표시, 저장 중 중복 제출 방지를 추가했습니다.
- 저장/조회 연동 범위를 `docs/prd/refentra-v1.1-reference-persistence.md`에 문서화했습니다.
