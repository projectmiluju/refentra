# 프로젝트 현황

**최종 업데이트:** 2026-03-27
**현재 버전:** v0.5.0
**배포 URL:** 미정

## 최근 변경
- `v0.5.0`에서 Redis 기반 실제 세션 인증이 추가됐습니다.
- 로그인은 더 이상 `localStorage` 플래그가 아니라 `httpOnly` 쿠키와 Redis Refresh Token으로 유지됩니다.
- `POST /api/v1/auth/login`, `POST /api/v1/auth/refresh`, `POST /api/v1/auth/logout`, `GET /api/v1/auth/me`가 추가됐습니다.
- 보호된 레퍼런스 API는 인증 사용자가 있어야만 접근할 수 있고, `uploader_id`도 세션 사용자 기준으로 결정됩니다.
- 첫 로그인 후 레퍼런스가 0건이면 단순 빈 상태 대신 온보딩 가이드를 보여주도록 변경했습니다.
