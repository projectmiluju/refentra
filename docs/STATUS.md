# 프로젝트 현황

**최종 업데이트:** 2026-03-26
**현재 버전:** v0.2.0
**배포 URL:** 미정

## 최근 변경
- 로그인 입력 검증과 인증 상태 저장을 추가해 비인증 사용자의 대시보드 접근을 차단했습니다.
- 대시보드에서 새 레퍼런스를 등록하는 모달 제출 흐름을 연결했습니다.
- 레퍼런스 API 입력값 trim 및 필수값 검증을 서버에 추가했습니다.
- Vitest/jsdom 기반 프론트엔드 테스트와 Go 핸들러 테스트를 추가했습니다.
- 로그인 화면 접근성, 한글 렌더링, 디자인 토큰 기준을 QA 결과에 맞춰 보강했습니다.
