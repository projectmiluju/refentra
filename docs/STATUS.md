# 프로젝트 현황

**최종 업데이트:** 2026-03-27
**현재 버전:** v0.5.1
**배포 URL:** 미정

## 최근 변경
- `v0.5.1`에서 Playwright 기반 브라우저 E2E가 추가됐습니다.
- 비로그인 보호 URL 접근, 로그인, 저장 후 새로고침 재조회, 로그아웃 후 재접근 차단이 실제 브라우저에서 자동 검증됩니다.
- 온보딩 빈 상태도 브라우저 시나리오에 포함됐고, 현재는 API 응답 mock으로 검증합니다.
- `npm test`가 `e2e` 및 `node_modules` 테스트를 잘못 수집하지 않도록 범위를 명시적으로 제한했습니다.
