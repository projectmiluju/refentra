# UI 컴포넌트 명세서 (Refentra v1)

**목적:** `build` 에이전트 및 프론트엔드 엔지니어가 UI 구현 시 바로 참조할 수 있는 컴포넌트 상세 스펙 문서.
**참고 디자인 시스템:** `docs/design/design-system.md`

## 1. 공통 버튼 (Button)
**역할:** 사용자 워크플로우 진행 및 액션 처리
**한글 및 접근성 고려사항:**
- 라벨 길이에 무관하게 최소 클릭 영역 보장을 위해 `min-width: 80px` 이상 적용
- 텍스트 오버플로우 방지를 위해 줄바꿈 금지 (`white-space: nowrap`)

| Prop | 타입 | 기본값 | 설명 |
|------|------|-------|------|
| variant | `primary` \| `ghost` | `primary` | `primary`는 CTA(저장, 로그인), `ghost`는 보조 অ্যাক션(취소 등) |
| size | `sm` \| `md` \| `lg` | `md` | `md` 기본 패딩, `lg`는 로그인 폼의 꽉찬 버튼 등에 사용 |
| isLoading | `boolean` | `false` | 로딩 상태 (스피너 표시 + 비활성화) |
| fullWidth | `boolean` | `false` | 가로로 완전히 100% 채움 여부 |

**상태별 스타일:**
| 상태 | 배경색 | 텍스트/보더 색상 | 기타 |
|------|-------|----------------|------|
| Primary 기본 | `#3B82F6` (Gradient 권장) | `#F8FAFC` | - |
| Primary 호버 | `#2563EB` | `#F8FAFC` | 밝기를 낮춰 피드백 제공 |
| Ghost 기본 | `transparent` | `#94A3B8` | - |
| 비활성(Disabled)| `#334155` | `#64748B` | `cursor: not-allowed` |

## 2. 기본 입력 필드 (Input / Textarea)
**역할:** 사용자 인증 정보 입력 및 레퍼런스 메타데이터(URL, 제목, 설명) 입력용

| Prop | 타입 | 설명 |
|------|------|------|
| type | `text` \| `password` \| `email` | 인풋 타입 지정 |
| isError| `boolean` | 유효성 검사 실패 시 `true`로 설정하여 경고색 표출 |
| placeholder| `string` | 입력 힌트 안내 텍스트 |

**상태별 인터랙션:**
| 상태 | 카드/배경색 | 텍스트 색 | 힌트(Placeholder) | 테두리 (Border) |
|------|-----------|---------|-----------------|---------------|
| 기본 | `#0F172A`(앱 배경) | `#F8FAFC` | `#94A3B8` | `1px solid #334155` |
| 포커스 | - | - | - | `1px solid #3B82F6` |
| 에러 | - | - | - | `1px solid #EF4444` |

## 3. 태그 칩 (Tag / Badge)
**역할:** 입력된 카테고리를 직관적으로 인지할 수 있는 작은 배지 UI

| 상태 | 배경색 | 텍스트 색 | 간격(Spacing) | 반경(Radius) |
|------|-------|---------|---------------|------------|
| 기본 | `#334155` (또는 투명도) | `#E2E8F0` | `px-2 py-1` | `4px` |

## 4. 레퍼런스 리스트 아이템 (Reference Row/Card)
**역할:** 대시보드 우측 메인 영역에 표시될 개별 레퍼런스

- **구조와 한글 오버플로우 정책:**
  - **제목 (Title):** 굵고 명확한 `Pretendard Bold` 텍스트. 너무 길면 한글 단어 단위 보존(`word-break: keep-all`)
  - **URL 링크:** 클릭 시 새 탭 이동 기능. 영문/주소 형태이므로 한 줄을 넘지 않게 자름 (`text-overflow: ellipsis; white-space: nowrap;`)
  - **부연 설명 (Description):** 최대 2~3줄까지만 보이고 말줄임표 처리 (`line-clamp`)
  - **태그 (Tags):** 태그 칩들이 가로 배열로 나열되며 여백 부족 시 줄바꿈 처리
  - **메타 (Meta):** 작성자 및 생성 일시 표기 (`#94A3B8`)

## 5. 프로젝트 한글화 컴포넌트 텍스트 매핑
Stitch 결과물이나 영문 베이스 템플릿을 실제 적용 시 아래의 네이밍을 100% 준수합니다:
- **인증 폼:** 로그인, 이메일 주소, 비밀번호, 비밀번호를 잊으셨나요?
- **공통 동작:** 취소, 저장하기, 더 보기
- **대시보드:** 새 레퍼런스 추가, 레퍼런스 검색...
